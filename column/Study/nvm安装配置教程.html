nvm 安装配置教程 ​

1、安装 ​

到 nvm 的 github 仓库下载 项目地址 下载完成之后将项目 zip 压缩包解压到无空格无中文的目录下：D:\tools\Nvm\nvm-noinstall

2、配置 ​

在 D:\tools\Nvm 目录中新建三个文件夹 nodejs_list、npm_global、npm_cache

  • nodejs_list: 激活的 nodejs 的存储路径
  • npm_global: 用来存放 npm 下载的全局包
  • npm_cache: 用来存放 npm 缓存文件

settings.txt 文件中参数解释：

  • root: nvm 的安装地址，即上文的%NVM_HOME%
  • path: 激活的 node.js 时的存储路径，即上文的%NVM_SYMLINK%
  • arch: 系统架构，如果你的电脑是 64 位，则填 64，如果是 32 位，则填 32
  • proxy: 代理配置
  • node_mirror: node.js 的下载源
  • npm_mirror: npm 的下载源

打开命令行，输入 nvm -v 显示版本号，即为安装配置成功

到此位置，nvm 的安装和配置完成
